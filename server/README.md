# Typeform interactive video


## Setup

Install the dependencies with `npm install`

## Local usage

- `npm start` Server will run on port 3001.
- `ngrok http 3001` You can use ngrok to create a tunnel. After that you need to modify your install.json file in repository 'cloudflare-app'  with ngrok address.
- `http://127.0.0.1:4040/inspect/http` - here you can inspect your traffic.
